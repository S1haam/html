HTML Global Attributes

Global attributes can be used on almost any HTML element. They add extra information or functionality.

id Attribute

The id attribute gives a unique identifier to an element.

class Attribute

The class attribute allows you to assign one or more classes to an element for styling or scripting.

lang Attribute

The lang attribute specifies the language of the element content.

dir Attribute

The dir attribute sets the text direction: ltr (left-to-right) or rtl (right-to-left).

hidden Attribute

The hidden attribute hides an element from view. It will not be displayed on the page.

tabindex Attribute

The tabindex attribute sets the order of keyboard navigation for focusable elements.

style Attribute

The style attribute allows you to add inline CSS styles directly to an element.
